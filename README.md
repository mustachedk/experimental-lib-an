# experimental-lib-an
Mustache Experimental Library for Android 

###### Add this to your root build.gradle
```bash
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```

###### Add this to your root build.gradle
```bash
dependencies {
	        implementation 'com.github.mustachedk:experimental-lib-an:1.0'
}
```
